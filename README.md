My project name is SportNest - Sports Facility Booking Management System
Project Overview

SportNest is a modern sports facility booking management platform designed for sports enthusiasts. The system allows users to discover, manage, and book sports facilities efficiently. Users can create facility listings, make bookings, update facility information, and manage their reservations through a user-friendly interface.

Live URL
https://sport-frontend-gray.vercel.app

Key Features
1.Fully responsive design for mobile, tablet, and desktop devices.
2.Secure authentication system with Email/Password and Google Sign-In.
3.Complete CRUD (Create, Read, Update, Delete) functionality for facility management.
4.JWT-based authorization for securing backend API routes.
5.Users can add, edit, and delete sports facilities.
6.Facility booking system with real-time booking management and make loading and not-found page.
7.Professional and user-friendly navigation bar and footer.
8.Proxy implementation for enhanced frontend security.
9.MongoDB database integration for efficient data storage and management.
10.Modern and attractive UI/UX design for an improved user experience .
